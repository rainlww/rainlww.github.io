💜 Neta 点赞工具

自动为标签下的作品点赞 + AI 生成留言

⚙️ 配置参数

要点赞的作品所在的标签名称
如果填写，只处理该角色的作品；留空则处理标签下所有作品
建议 20-50 个，避免 API 限流
可用占位符：{title}（作品标题）、{author}（作者）、{tags}（标签）、{desc}（描述）
为热度前 N 的作品留言（热度 = 点赞数）

📖 使用说明

  • 配置参数：填写标签名、角色 UUID（可选）等信息
  • 生成脚本：点击按钮自动生成 bash 脚本
  • 复制脚本：将生成的脚本复制到本地文件（如 like-bot.sh）
  • 执行脚本：在终端运行 bash like-bot.sh
  • 查看报告：脚本执行完毕后会生成 HTML 报告
